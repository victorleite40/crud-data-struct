# CRUD Data Struct
Simple CRUD implementation in java.

### API

#### Add
```add <id> <first_name> <last_name> <birtday> <phone_number>```

#### Delete
```del <id>```

#### Get
```info <id>```

#### Query
```query <tag>:<value> ...```

##### Tags:
`fn = First Name`
`ln = Last Name`
`bd = Birth Day`
`pn = Phone Number`
