# CRUD Data Struct
Simple CRUD implementation in java.

`
Create, Read, Update and Delete.
`

## API

### Add
```
add <id> <first_name> <last_name> <birtday> <phone_number>
```

### Delete
```
del <id>
```

### Get
```
info <id>
```

### Query
```
query <tag>:<value> ...
```
##### Tags:
`fn = First Name` <br>
`ln = Last Name` <br>
`bd = Birth Day` <br>
`pn = Phone Number` <br>
